One-to-Multi Agent - Database ER図

プロジェクト概要

One-to-Multi Agentは、テキスト、音声、動画から複数のSNSプラットフォーム向けコンテンツを自動生成するアプリケーションです。

このデータベースは、ユーザー管理、コンテンツ管理、履歴管理の3つの主要な機能をサポートしています。

📝 注目ポイント

  • 🆕 プロンプト管理: 3階層プロンプト（キャラクター全体/媒体別/都度設定）
  • 🆕 画像生成機能: キャラクター画像設定と生成画像管理
  • 🆕 履歴機能強化: 使用プロンプトと生成画像の履歴保存
  • 二重管理: content_metadata/platform_content と content_history/platform_contents で類似データを管理
  • ファイル管理: content_metadataにアップロードファイル情報を保存
  • プラットフォーム対応: Instagram, TikTok, Threads, X (Twitter), YouTube, Blog向けコンテンツ生成
  • ユーザー認証: Google OAuth 2.0による認証システム
